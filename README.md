# Configurações do `settings.json`

Este arquivo contém as configurações personalizadas para o ambiente de desenvolvimento no Visual Studio Code, otimizado para produtividade e qualidade de código.

## Configurações Gerais do Editor
- Tamanho da fonte e altura de linha configurados para melhor legibilidade.
- Régua de código definida em 80 e 120 caracteres para orientação visual.

## Configurações Específicas de Arquivos
- Associações de tipos de arquivo para suporte a sintaxe, como `.env`, `.prettierrc`, e `.css`.

## Sugestões e Imports Automáticos
- Importações automáticas habilitadas para JavaScript e TypeScript.
- Atualização automática de imports ao mover arquivos.

## Formatação Automática e Validador de Código
- Formatação automática ao salvar, integrada com ESLint para validação de JavaScript e TypeScript.

## Configurações de Tema e Ícones
- Tema de cores definido como `Min Dark`.
- Conjunto de ícones configurado como `material-icon-theme`.

## Configurações do Terminal e Git
- Configurações específicas para o terminal e integração com Git, incluindo ajustes de fonte e comportamento.
